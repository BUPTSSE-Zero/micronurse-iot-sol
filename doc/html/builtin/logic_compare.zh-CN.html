比较函数

相等

描述

两个输入相等，输出true，反之false

输入

in1: 任意类型

in2: 任意类型

输出

bool: 布尔值 bool = (in1 == in2)

大于

描述

in1 大于 in2，输出true，反之false

字符串的比较是比较首字符的编码大小

输入

in1: 数值或字符串

in2: 数值或字符串

输出

bool: 布尔值 bool = (in1 > in2)
