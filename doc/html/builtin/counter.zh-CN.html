计数器

计数

描述

此服务提供计数功能。

输入

trigger: 任意类型 触发一次计算

reset: 布尔类型 重置计数

输出

out: 任意类型 计数器当前值

倒数

描述

此服务提供倒数计数功能。

输入

trigger: 任意类型 触发一次计算

reset: 布尔类型 重置计数

orig: 整型 重置时使用的初始值

输出

hit: 布尔类型 当且仅当计数器减到0时，hit输出true
